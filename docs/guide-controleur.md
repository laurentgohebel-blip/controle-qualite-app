# Guide d'utilisation — Contrôleur

Application : **Contrôle Qualité Propreté**
Profil : **Contrôleur** (et **Manager** — les fonctions de validation sont précisées)

Le contrôleur est sur le terrain. Son rôle : saisir les contrôles et les soumettre pour validation.

---

## 1. Premier accès — créer mon compte

1. Votre manager vous a transmis une URL d'application + un email d'invitation.
2. Ouvrez l'URL sur votre téléphone.
3. Cliquez **« Créer un compte »**.
4. Renseignez :
   - **Email** : exactement celui que votre manager a indiqué
   - **Mot de passe** de votre choix (6 caractères min)
   - **Laissez vide** le champ "Nom de société" — vous rejoignez l'organisation existante
5. **« Créer mon compte »** → vous êtes connecté avec votre rôle.

### Installer l'app sur mobile (PWA, recommandé pour le terrain)

- **Android (Chrome)** : ⋮ → Ajouter à l'écran d'accueil
- **iPhone (Safari)** : Partager → Sur l'écran d'accueil

L'app s'ouvre alors en plein écran comme une vraie application.

---

## 2. Démarrer un contrôle

### Depuis un contrôle planifié (le plus courant)

1. Menu **Planning** → repérez votre contrôle (calendrier ou liste "À venir")
2. Bouton **« Démarrer »**

### Sans planification (contrôle inopiné)

1. Tableau de bord → **« Nouveau contrôle »**
2. Choisissez le site et le type :
   - **Programmé**
   - **Inopiné**
   - **Contradictoire client**

---

## 3. La saisie en 3 étapes

### Étape 1 — Informations

- Site, date, type, **agent évalué**
- **Cochez les locaux à contrôler** parmi ceux du site
- Astuce : si vous avez déjà contrôlé ce site, les locaux du dernier contrôle sont **pré-cochés automatiquement**

### Étape 2 — Évaluation par local

Pour chaque local sélectionné :

Pour chaque critère qui s'applique, choisissez la note :

| Note | Couleur | Quand l'utiliser |
|---|---|---|
| **Conforme** | Vert | Tout est OK |
| **Partiellement conforme** | Jaune | Un défaut mineur |
| **Non conforme** | Rouge | Problème significatif → génère une action corrective auto |
| **Non applicable** | Gris | Ce critère ne concerne pas ce local |

- **Commentaire** : zone libre pour préciser (recommandé sur les NC)
- **Photo 📷** : prend une photo (utilise l'appareil de votre téléphone) — indispensable pour justifier une NC

Quand tous les critères sont notés → **« Local suivant → »** ou **« Continuer → »** au dernier local.

### Étape 3 — Récapitulatif

- Le **taux de conformité** est calculé automatiquement (pondéré par les coefficients des critères)
- Ajoutez un **commentaire général** si besoin
- **« Valider le contrôle ✓ »** → enregistré

---

## 4. Signatures électroniques

Sur la fiche du contrôle, en bas, carte **Signatures** :

1. Vous signez dans la zone **« Contrôleur »** (au doigt sur tablette/téléphone, à la souris sur ordinateur)
2. Faites signer l'agent évalué dans la zone **« Agent évalué »**
3. **« Enregistrer les signatures »** → le contrôle passe en statut **Validé**

---

## 5. Workflow de validation (si Manager)

**En tant que Contrôleur** :
- Une fois le contrôle Terminé → bouton **« Soumettre pour validation »**
- Votre manager le voit en statut "À valider" et l'approuve ou le rejette
- En cas de rejet, le motif s'affiche dans le commentaire général → vous le retrouvez en "Brouillon" pour reprise

**En tant que Manager** :
- Menu **Contrôles** → filtre statut **« À valider »**
- Ouvrez le contrôle → vérifiez → **« ✓ Valider »** ou **« ✗ Rejeter »** (avec motif)

---

## 6. Envoyer le rapport au client

Une fois le contrôle Validé, sur sa fiche :

- **« Exporter PDF »** : télécharge le rapport pour archivage
- **« 📧 Envoyer »** : ouvre votre client mail avec destinataire (email client du site) + objet + corps pré-remplis. Le PDF est téléchargé pour être joint manuellement à l'email.
- **« 🔗 Lien client »** : génère un lien web sécurisé (valide 90 jours) automatiquement copié dans votre presse-papier. Collez-le dans un mail/SMS → le client clique → voit le rapport sans s'inscrire.

---

## 7. Suivre les actions correctives

Menu **Actions** :
- Liste de toutes les actions générées par vos NC
- Tableau de bord en haut : Total / Ouvertes / En cours / Soldées / **En retard** (urgent !)
- Cliquez sur un statut (Ouverte / En cours / Soldée) pour le changer en 1 clic
- **« Modifier »** : éditer description, action à mener, échéance
- **« Notifier »** : si le responsable de l'action a un email, ouvre votre mail avec un message pré-rempli (description, action, échéance)

## 8. Mode hors-ligne

Si vous perdez le réseau (sous-sol, parking…) :
- **Bandeau orange** en haut de l'écran
- Votre **saisie est sauvegardée automatiquement** sur votre téléphone à chaque modification
- **Important** : ne fermez pas l'onglet tant que vous êtes hors-ligne. Reprenez la connexion (4G/wifi) avant de valider le contrôle, sinon vous risquez de perdre vos données.
- Si vous reprenez la connexion, votre saisie est restaurée à l'ouverture du contrôle planifié.

---

## 9. Astuces terrain

**Gagner du temps sur la saisie**
- Les **locaux du dernier contrôle** du site sont pré-cochés à l'étape 1
- L'**agent évalué** du dernier contrôle est pré-sélectionné
- Si tous les locaux sont conformes : laissez les notes par défaut (Conforme), commentez juste si nécessaire

**Justifier une NC**
- **Toujours prendre une photo** sur Non conforme → preuve incontestable
- **Toujours commenter** : "Trace de boisson à l'angle nord-est" plutôt que "sale"

**Sur tablette plutôt que téléphone**
L'interface est responsive mais une tablette (8-10") rend la saisie beaucoup plus confortable, surtout sur l'écran d'évaluation par local.

## 10. FAQ Contrôleur

**Je ne vois pas un site dans la liste.**
Demandez à votre admin de vous rattacher à l'organisation, ou vérifiez que vous êtes bien connecté avec le bon email.

**La photo ne fonctionne pas.**
Autorisez l'accès à l'appareil photo dans les paramètres du navigateur (icône cadenas à gauche de l'URL).

**J'ai perdu un contrôle en cours.**
Si vous étiez à l'étape 2 ou 3, votre brouillon est sauvegardé. Recliquez "Nouveau contrôle" sur le même site → la saisie revient. Sinon, contactez votre manager.

**Comment savoir si le client a bien reçu le rapport ?**
L'app ne suit pas la réception des mails. Mieux : envoyez plutôt un **🔗 Lien client** — vous voyez côté client s'il l'a ouvert (à venir dans une future version).

**Que se passe-t-il après "Soumettre pour validation" ?**
Votre manager reçoit le contrôle en statut "À valider". Il vérifie, valide (ou rejette avec motif). Une fois validé, le rapport peut être envoyé au client.
